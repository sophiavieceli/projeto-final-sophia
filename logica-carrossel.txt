Inicialmente todos com display none, menos o início

A partir do início:
- Se clicar em "Entrar":
    Início display none
    Entrar display flex
- Se clicar em "Criar conta":
    Início display none
    Cadastro display flex

A partir de entrar:
- Se clicar em voltar:
    Início display flex
    Login display none
- Se clicar em não tem uma conta:
    Login display none
    Cadastro display flex

A partir de cadastro:
- Se clicar em voltar:
    Início display flex
    Cadastro display none
- Se clicar em já tem uma conta:
    Cadastro display none
    Login display flex
- Se clicar em próximo:
    Cadastro display none
    Configurações do perfil display flex

A partir de configurações do perfil:
- Se clicar em voltar:
    Configurações do perfil display none
    Cadastro display flex
- Se clicar em próximo:
    Configurações do perfil display none
    Configurações de idioma display flex

A partir de configurações de idioma:
- Se clicar em voltar:
    Configurações de idioma display none
    Configurações do perfil display flex
- Se clicar em próximo:
    Configurações do idioma display none
    Configurações de interesses display flex

A partir de configurações de interesses:
- Se clicar em voltar:
    Configurações de interesses display none
    Configurações de idioma display flex
- Se clicar em pronto:
    Configurações de interesses display none
    Confirmação de cadastro display flex
